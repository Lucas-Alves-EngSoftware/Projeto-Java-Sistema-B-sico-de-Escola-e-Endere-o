# Projeto-Java-Sistema-B-sico-de-Escola-e-Endere-o
Este projeto demonstra uma estrutura simples em Java contendo duas classes principais: Escola e Endereco, utilizadas para representar informações básicas sobre uma instituição de ensino e sua localização.  A organização segue boas práticas iniciais de POO (Programação Orientada a Objetos), ideal para estudos e evolução futura para APIs REST.
